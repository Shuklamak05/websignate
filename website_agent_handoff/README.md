# Signate Website Agent Handoff

This package contains organized content and images extracted from the two brochure PDFs in the source directory.

## Contents

- `products.json` - structured product catalogue with web-friendly copy, benefits, pack sizes where readable, image paths, and source page references.
- `product_panels/` - cropped brochure panels for individual products or sections.
- `source_page_images/` - full-resolution page images extracted from the PDFs.
- `cattle_contact_sheet.jpg` and `companion_animals_contact_sheet.jpg` - quick visual overview sheets.

## Important Notes

The PDFs are image-based, not text PDFs. The product data in `products.json` was transcribed/summarized from the brochure images and should be verified by a human before publishing medical, dosage, diagnostic, or nutritional claims.

Recommended use for website development:
1. Use `products.json` for page structure, product cards, collection pages, and SEO draft copy.
2. Use `product_panels/` as immediate visual references or temporary product images.
3. Use `source_page_images/` to crop higher-quality final assets or validate exact brochure wording.
